## 📝 Requerimientos básicos

- Node.js v14.17.0 or higher ([Download](https://nodejs.org/es/download/))
- YARN v1.22.17 or higher
- NPM v6.14.13 or higher

<a name="scripts"></a>

## 💻 Scripts

Instalar dependencias del monorepo

```
yarn install
```

Transpilar spa

```
yarn build
```

Test

```
yarn test
```
